# LoginModule 登陆demo

项目技术：
> * 开发工具: IntelliJ IDEA 2017
> * JDK版本: 1.7
> * 项目构建工具: Maven 3.5.2
> * 应用服务器: Tomcat 7.0
> * 前端框架: Bootstrap 3.3.7, Jquery 1.12.4
> * 后端框架: Spring 4.2.4, SpringMVC 4.2.4, Mybatis 3.4.4
